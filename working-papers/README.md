# Working Papers

Working papers are developed philosophical arguments in manuscript form. They represent substantial philosophical work ready for critical engagement, though not necessarily in their final form.

---

## Current Papers

### [Motivational Competition: A Philosophical Proposal](motivational-competition.md)

Explores how multiple primary motivations simultaneously compete to produce a single realized behavior, introducing a framework of three primary motivational systems (self-preservation, duty, meaning) and secondary modifier systems.

### [Motivational Analysis of the Kolyma Camps](kolyma-analysis.md)

Applies the motivational framework to Varlam Shalamov's accounts of the Kolyma labor camps, examining how the framework explains human behavior under conditions of extreme deprivation.

## Current Project

### Interpretive Motivational Systems Framework (IMSF)

**Objective:** To develop a descriptive philosophical account of how competing motivations produce a single realized behavior.

**Current Stage:** Working Paper (Version 0.1)

**Current Questions:**
- Are there exactly three primary motivational systems?
- Is the distinction between primary and secondary systems coherent?
- How should motivational weighting be understood?
- Which existing philosophical theories offer the strongest objections?
